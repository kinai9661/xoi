# AI 圖片生成工具

一個專業的 AI 圖片生成工具，支持多種先進的圖片生成模型。

## 功能

- 🤖 支持多個 AI 模型
- 🎯 靈活的參數配置
- 🚀 快速響應和高質量輸出
- 💻 友好的 Web 界面

## 使用方法

1. 輸入圖片描述
2. 選擇模型和參數
3. 點擊「生成圖片」按鈕
4. 等待結果

## 環境變數

需要設置以下環境變數：

- `API_KEY`: 你的 API 密鑰

在 HuggingFace Spaces 中，通過 Repository secrets 設置。
